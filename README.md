# Todo-List
Todo List using Python and MySQL. 

## Requirements

- tkinter
- mysql.connector
- datetime

## Features

The application allows you to:

  - Add new tasks with a due date.
  - View the list of all the tasks, sorted by due date if wanted.
  - Delete tasks.
  - Mark tasks as completed.
